# Node.js CRUD API

A simple REST API built with Node.js, Express.js, MongoDB, and Mongoose.

## Features

- Create Person
- Get All Persons
- Get Person by ID
- Update Person
- Delete Person

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Installation

```bash
npm install
```

## Run Project

```bash
node index.js
nodemon index.js
```

Server runs on:

```text
http://localhost:3000
```
